# FazPeloZap — Landing Page

Landing page for **FazPeloZap**, a SaaS tool that helps sellers create Shopee listings directly through WhatsApp.

🌐 **Live:** [www.fazpelozap.com.br](https://www.fazpelozap.com.br)

---

## About

FazPeloZap lets sellers create product listings on Shopee without ever touching the Shopee seller panel. The user sends product photos via WhatsApp, answers a few guided questions, and the listing is created automatically.

## Tech Stack

- Plain HTML + CSS (no framework, no build step)
- [Poppins](https://fonts.google.com/specimen/Poppins) via Google Fonts
- [Font Awesome 6](https://fontawesome.com/) for icons
- Hosted on **GitHub Pages**

---

## Local Development

No build tools required. Just open `index.html` in your browser:

```bash
open index.html
```

Or serve it locally with any static server:

```bash
npx serve .
```
